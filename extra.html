WURFL

An online database for devices

What is WURFL?

This is pronounced “wuhr-full”. It translates to Wireless Universal Resource File. It is an open-source device database with a lot of information about specific capabilities of mobile devices and their browsers.

WURFL offers multiple capabilities for their commercial WURFL.js Basic, Standard, or Pro licenses. Here is a description of 3 of those capabilities offered:

modem

Returns the name of the device’s modem. Eg. “Snapdragon X60 5G”. Note: The modem capability will return “Integrated” if the modem is not a separate module and is integrated into the chipset. In those cases the return value of the chipset capability will be the modem’s make and model.

complete_device_name

Concatenation of brand name, model name, and marketing name (where available) of a device into a single string.

is_mobile

Returns true if the device is wireless (e.g., Wi-Fi, 3G, LTE). Specifically, a tablet or mobile phone is considered wireless. Otherwise, a desktop or laptop is false.
